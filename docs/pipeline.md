# Analysis Pipeline

1. Repo Ingestion  
   Takes a GitHub repository and extracts relevant source files.

2. Static Code Analysis  
   Analyzes code structure without understanding business logic.

3. Structural Summarization  
   Converts raw analysis into architecture-level facts.

4. Architecture Reasoning  
   Uses an LLM to reason about design quality and tradeoffs.

5. Report Generation  
   Produces a human-readable architecture review.
